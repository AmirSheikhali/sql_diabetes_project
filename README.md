# sql_diabetes_project
A web application that allows users to track their daily health metrics related to diabetes, such as  blood glucose levels, carbohydrate intake, insulin dosage, and notes. The app helps users visualize trends and better understand their health over time

This application includes three preloaded users for demonstration purposes:
Amir – Serves as a primary example user used to showcase the interface design and data visualization.
David and Jimmy – Interactive demo users that allow testing of functionality such as switching profiles, viewing different datasets, and observing how charts update dynamically.
All users are part of the demo dataset and help illustrate how the application handles multiple profiles and health data tracking.


⚠️WARNING⚠️ : This app is for educational purposes only and is NOT intended to provide medical advice or be used for diagnosis or treatment.
